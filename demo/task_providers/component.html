This provider provides tasks related to mindfully attending to topics. Often, mindful attention towards a topic over a sustained period of time can be more effective than working on an existing task that might not be relevant, or might not be the highest-priority thing.

Registering topics

In order to know which topics to create tasks for, topics must be registered with the provider via a `RegisterTopicsRequest`. Development of this functionality is still in progress.
