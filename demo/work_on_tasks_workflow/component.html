cyber-ui-work-on-tasks-workflow

cyber-ui-work-on-tasks-workflow handles retrieving and displaying a list of task suggestions.

Inputs

  • displayComponent: the Angular component to use to display the tasks. Tasks will be passed to the display component's tasks input as an Observable<TASK_T[]>.

Usage notes

  • You'll probably want to constrain the width of this workflow, to maintain aesthetically reasonable aspect ratios of the interior items such as expansion panels (unless those items have their own max widths.) For example, you could give this workflow max-width: 50em;
