cyber-ui-work-on-tasks-workflow

cyber-ui-work-on-tasks-workflow handles retrieving and displaying a list of task suggestions using an arbitrary display component.

Inputs

  • displayComponent: the Angular component to use to display the tasks. Tasks will be passed to the display component's tasks input as an Observable<TASK_T[]>.

Example

This example uses the cyber-ui-task-accordion component as the display component (but you could provide your own custom display component.)
