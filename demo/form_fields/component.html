cyber-ui-form-fields

cyber-ui-form-fields renders a list of fields and binds them to a data model. This avoids the boilerplate of coding form templates by hand throughout your app. It's also useful for rendering dynamic field lists.

Outputs

  • The change event emits whenever any of the fields' values have changed
  • The save event emits when the user has initiated a save, such as by pressing enter on a focused field when the saveOnEnter option is set in the field's config
